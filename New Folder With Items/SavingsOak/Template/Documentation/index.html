JavaScript and Ajax

This template include 11 javascript files:.

  • bootstrap.min.js -scripts for bootstrap framework
  • html5shiv.min.js -this script is the defacto way to enable use of HTML5 sectioning elements in legacy IE
  • jquery.ajaxchimp.min.js -AjaxChimp is a jQuery plugin that lets you ajaxify your mailchimp form.
  • jquery.easing.js - script give advanced easing options
  • jquery.scrollUp.min.js -scripts for create a customisable 'Scroll to top'
  • jquery-1.12.4.min.js -JavaScript library
  • magnific-popup.min.js -responsive lightbox
  • owl.carousel.min.js -script for responsive carousel slider
  • modernizr.js -JavaScript library that detects HTML5 and CSS3 features in the user's browser
  • respond.min.js -a fast and lightweight polyfill for min/max-width CSS3 Media Queries
  • wow.min.js -script for reveal animations when you scroll
  • main.js -common js scripts for this template

All code are beautifully written. You can customize easily. Just look the js code.
Example: If you want to Configure Mailchimp, just find the following block and change the post url:
 

	


/*----------------------------------------
 Newsletter Subscribe
 --------------------------------------*/

   $(".subscribe-mail").ajaxChimp({
        callback: mailchimpCallRep,
        url: "mailchimp-post-url" //Replace this with your own mailchimp post URL. Just paste the url inside "".
    });

function mailchimpCallRep(resp)
{
    if (resp.result === "success") {
        $(".sucess-message").html(resp.msg).fadeIn(1000);
        $(".error-message").fadeOut(500);
    } else if (resp.result === "error") {
        $(".error-message").html(resp.msg).fadeIn(1000);
    }
}
